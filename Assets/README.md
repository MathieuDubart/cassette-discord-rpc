# Assets

Place `cassette-icon.png` in this directory before building.

This icon is used as the small image in Discord Rich Presence (`small_image`).
It must be committed to the repository so the raw GitHub URL used in the activity payload resolves correctly:

```
https://raw.githubusercontent.com/MathieuDubart/cassette-discord-rpc/main/Assets/cassette-icon.png
```
